# Contributing

Thanks for your interest in contributing to migux.

## How to contribute

- Open an issue to discuss changes before large work.
- Keep changes focused and incremental.
- Add or update documentation when behavior changes.

## Development

- Rust edition: 2024
- Format: `cargo fmt`
- Lint: `cargo clippy` (optional)
- Build: `cargo check` or `cargo build`

## Commit style

Use short, clear messages (English preferred), e.g.

- `feat(proxy): stream chunked responses`
- `fix(core): handle keep-alive timeouts`

## Reporting issues

Please include:

- what you expected
- what happened
- steps to reproduce
- logs (if relevant)

## Code of Conduct

Be respectful and constructive.
